# Physical AI & Humanoid Robotics Course Documentation

Welcome to the documentation for the Physical AI & Humanoid Robotics course. This comprehensive educational resource is designed to guide students through the fascinating world of embodied artificial intelligence and humanoid robotics.

## Course Overview

This course covers the following major topics:

1. **Module 1: The Robotic Nervous System (ROS 2)**
   - ROS 2 fundamentals: nodes, topics, and services
   - Python integration with rclpy
   - URDF modeling for humanoid robots
   - Practical exercises integrating concepts

2. **Module 2: Digital Twin Simulation**
   - Gazebo physics simulation
   - Collision detection and environmental physics
   - Unity integration for advanced rendering
   - Simulated sensors and perception

3. **Module 3: NVIDIA Isaac AI Brain**
   - Isaac Sim for advanced simulation
   - Isaac ROS for perception and navigation
   - SLAM and Nav2 systems
   - AI perception techniques

4. **Module 4: Vision-Language-Action**
   - Whisper voice command integration
   - Large language model integration for planning
   - Action generation from high-level goals
   - Capstone autonomous humanoid project

## Technical Requirements

This course uses:

- ROS 2 Humble Hawksbill or later LTS version
- Python 3.10+ with robotics-specific packages
- Gazebo simulation environment
- NVIDIA Isaac tools (where applicable)
- Ubuntu 22.04 LTS or equivalent development environment
